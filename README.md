Site web interactif permettant de découvrir et réserver des voyages dans le temps, avec trois destinations phares : Paris 1889, Crétacé, et Florence 1504. Ce projet a été développé avec HTML, Tailwind CSS et JavaScript, en utilisant des visuels générés par IA pour une expérience immersive.

Hero Section Vidéo : animation de fond 
Galerie des destinations : Cards interactives pour les 3 époques
Chatbot IA (Chronos)
Réservation : Formulaire avec validation des données.


Technologies 

HTML / CSS
Tailwind CSS 
JavaScript (Vanilla)
AOS Library pour les animations
